Pagespeed Insights to Google Sheets

You will need to create service accounts and enable both Google Drive API and Google Pagespeed Insights API

https://developers.google.com/speed/docs/insights/v5/get-started

https://developers.google.com/drive/api/v3/about-sdk

1. Enable both APIs and create service account
2. Create a new Google Sheet and share with service account Email
3. Update script with Sheet name and worksheet names
